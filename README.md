# Employee Attrition Prediction using Machine Learning

## Project Overview

This project aims to predict employee attrition using machine learning techniques and provide actionable insights for HR teams. Employee attrition can significantly impact recruitment costs, productivity, and workforce planning. By analyzing employee-related factors, this project identifies patterns associated with employee turnover.

## Dataset

* IBM HR Analytics Employee Attrition Dataset
* Total Records: 1,470 employees
* Target Variable: Attrition (Yes/No)

## Project Tasks

### 1. Data Loading & Exploration

* Dataset inspection
* Attrition rate calculation
* Identification of numeric and categorical features

### 2. Data Cleaning & Preprocessing

* Missing value analysis
* Removal of irrelevant columns
* One-Hot Encoding
* Feature Scaling

### 3. Exploratory Data Analysis

* Attrition by Department
* Attrition by Job Role
* Attrition vs Monthly Income
* Attrition vs Work-Life Balance
* Attrition vs Years at Company

### 4. Machine Learning Models

* Logistic Regression
* Random Forest Classifier
* Gradient Boosting Classifier

### 5. Model Evaluation

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC Score
* Confusion Matrix

### 6. Key Findings

* Monthly Income was the most influential feature.
* Age and Overtime were strong predictors of attrition.
* Sales Representatives showed the highest attrition rate.
* Lower income and excessive overtime increased attrition risk.

### 7. Business Recommendations

* Improve retention strategies for Sales employees.
* Reduce excessive overtime where possible.
* Enhance employee engagement and career development programs.
* Monitor high-risk employees proactively.
